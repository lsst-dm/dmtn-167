Path Definition Group Access Controls Notes
/repo/main -> /repo/main_$BEST Read-only. This is a symlink to the latest/best data repository for all non-simulated data.
/repo/main_$DATE Gen3 data repository for all non-simulated data. Read-only.
/repo/main_$DATE/u Root for personal collections. All users can create user-owned subdirectories matching their usernames. If possible, make username subdirectories recursively world-readable by default, and disable creation of non-username subdirectories or files here.
/repo/main_$DATE/runs Root for shared instrument-generic processing outputs. operators Group members can create user-owned subdirectories. If possible, make created subdirectories group-writeable and world-readable (recursively), and disable creation of files here.
/repo/main_$DATE/$INSTRUMENT/runs Root for shared instrument-specific processing outputs. operators Group members can create user-owned subdirectories. If possible, make created subdirectories group-writeable and world-readable (recursively), and disable creation of files here.
/repo/main_$DATE/$INSTRUMENT/calib Root for shared calibration production outputs. cpp Group members can create user-owned subdirectories. If possible, make created subdirectories group-writeable and world-readable (recursively), and disable creation of files here.
/repo/main_$DATE/refcats Root for ingested reference catalogs. refcats Group members can create user-owned subdirectories. No permanent group members; users granted temporary access upon request.
If possible, make created subdirectories group-writeable and world-readable (recursively), and disable creation of files here.
/repo/main_$DATE/skymaps Root for SkyMap-definition datasets. skymaps Group members can create user-owned files. No permanent group members; users granted temporary access upon request. If possible, disable creation of directories here.
/repo/DC2 -> /repo/DC2_$BEST Read-only. This is a symlink to the latest/best data repository for all DC2 data.
/repo/DC2_$DATE Gen3 data repository for DC2. Read-only.

/repo/DC2_$DATE/runs Root for shared processing outputs (mixed simulation versions). operators Group members can create user-owned subdirectories. If possible, make created subdirectories group-writeable and world-readable (recursively), and disable creation of files here.
/repo/DC2_$DATE/$X.$Y[ip]/calib Gen3-produced calibrations to be used with this DC2 version. cpp Group members can create user-owned subdirectories. If possible, make created subdirectories group-writeable and world-readable (recursively), and disable creation of files here.
/repo/DC2_$DATE/refcats Root for ingested reference catalogs. refcats Users granted temporary access upon request. No permanent group members; users granted temporary access upon request.
If possible, make created subdirectories group-writeable and world-readable (recursively), and disable creation of files here.
/repo/DC2_$DATE/skymaps Root for SkyMap-definition datasets. skymaps Group members can create user-owned files. No permanent group members; users granted temporary access upon request. If possible, disable creation of directories here.
